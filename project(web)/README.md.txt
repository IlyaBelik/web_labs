1. встановлюю node.js і mongodb(вставнолюю path)
2. у cmd, go to project directory.
3. npm (npm init), 
 nodemon (npm install --save-dev nodemon),
 express (npm install express --save),
 mongoose (npm install mongoose --save),
 cors (npm install cors)
4. Встановлюю https://addons.mozilla.org/en-US/firefox/addon/cors-everywhere/ (або альтернативу у будь-якому браузері)
5. У папці з проектом виконую md "/data/db/"
6. У папці з проектом запускаю  "mongod" і "npm run start".
7. перевіряю http://127.0.0.1:8000/ , localhost port: 8000/feedbacks
