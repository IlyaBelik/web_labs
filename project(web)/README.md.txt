1. Install node.js ³ mongodb(set path)
2. in cmd, go to project directory.
3. npm (npm init), 
 nodemon (npm install --save-dev nodemon),
 express (npm install express --save),
 mongoose (npm install mongoose --save),
 cors (npm install cors)
4. Install https://addons.mozilla.org/en-US/firefox/addon/cors-everywhere/ 
5. In project directory do: md "/data/db/"
6. In project directory start  "mongod" ³ "npm run start".
7. Check http://127.0.0.1:8000/ , localhost port: 8000/feedbacks
